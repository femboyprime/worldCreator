# worldCreator
> createur de monde *pour la 1.8.9* <br></br>.
> **mon 2e projet en java. <br><sub>mieux organisé lui**</sub></br>

## Commande:
- **/wc**
  - create / cr / c
    - crée une map.
  - teleport / tp / t
    - teleport a la map crée.
  - menu / m
    - ouvre le menu de configuration pour la generation de map.
